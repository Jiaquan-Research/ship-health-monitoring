# Dataset Registry

This registry records datasets already mentioned or used within the project. It is a navigation and status document only.

It does not certify dataset quality, licensing, suitability, or final validation status.

## LBNL Chiller Plant

Dataset Name

LBNL Chiller Plant

Domain

HVAC / Chiller Plant

Source

LBNL dataset used by the current validation pipeline.

Real / Simulation

Real operational dataset.

Evidence Level

Experimentally used in the current project.

Current Usage

Primary validation pipeline for Expected State, Residual Statistical Validity, Semantic Admissibility, Semantic Window validation, and Health Evidence prototype benchmarking.

Validation Status

Validated for current framework development up to Evidence Prototype within the LBNL operating context.

Future Role

Pipeline validation reference dataset.

Notes

Does not contain confirmed degradation labels or maintenance-event annotations. It shall not be treated as direct validation for marine systems.
