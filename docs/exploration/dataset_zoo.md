# Industrial Dataset Zoo

Observation Ledger

---

## 1. Purpose

This document records datasets available for exploratory work.

Goals:

- observation
- curiosity
- diversity

Not:

- validation
- evidence generation
- theory support

Positive and negative observations are equally valuable.

Failure is acceptable.

---

## 2. Dataset Zoo Rules

No dataset is expected to support Q0.

No single dataset can validate or invalidate Q0.

Positive observations do not imply evidence.

Negative observations are informative.

Cross-dataset comparison should remain qualitative.

No ranking is intended.
