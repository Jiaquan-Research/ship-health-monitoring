# Engineering Variable Taxonomy

This document classifies the LBNL dataset variables by engineering meaning.

This is a semantic taxonomy only. It is not a statistical analysis, not Condition Variable selection, and not a feature selection result.

Classification uses dataset column names, Data Quality Audit column inventory, HVAC/chiller naming conventions, and engineering semantics.

Purpose

## Controller Output

Purpose

Control actions generated by the control system.

Engineering meaning

These variables represent controller responses.

They may contain both operating-condition information and embedded equipment-health information.

Their suitability for the Condition Space shall be evaluated later.

No decision is made in this task.

Variables

* CHL_COMP_SPD_CTRL_1
