# Inventory Audit Outputs

This directory contains the full project inventory and consistency audit generated as read-only reference material for reviewer-facing documentation.

- [timeline_full.md](timeline_full.md) reconstructs the full git timeline chronologically, classifying each commit by project line and outcome and flagging mixed or potentially inconsistent history items.
- [number_provenance.md](number_provenance.md) inventories R2/R-squared, Spearman, Cohen's d, p-value, ratio, lifecycle, and detection-related numbers found in `docs/` and `outputs/`, with producing script and input provenance where it could be located.
- [consistency_check.md](consistency_check.md) compares the Claim Registry, attack review summary, project status files, exploration review, and latest validation outputs, highlighting stale or conflicting status language.
- [asset_inventory.md](asset_inventory.md) lists project assets under `docs/`, `src/`, and `outputs/`, with short descriptions and ACTIVE / SUPERSEDED / ORPHANED flags.
